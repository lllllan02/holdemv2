📋 如何使用这些静态页面

  1. 直接打开： 双击任意HTML文件即可在浏览器中查看
  2. 本地服务器： 使用 python -m http.server 或 npx serve 启动本地服务器
  3. 开发环境： 在项目根目录运行 npm run dev 启动完整的开发环境
  4. 构建版本： 运行 npm run build 生成生产版本
